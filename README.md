## 🛠️ Tech Stack

- **Framework**: Next.js 16 (Static Export) + React 19 + TypeScript
- **Styling**: Tailwind CSS v4 + Custom CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Font**: DM Sans + DM Serif Display

## 🚀 Getting Started (Local Dev)

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000/finwise](http://localhost:3000/finwise) to view locally.

---

## 📦 Build & Deploy

```bash
# Production build (static export → /out folder)
npm run build
```

Deployment is **automated via GitHub Actions** — just push to `master` branch and it deploys to GitHub Pages automatically.

### Setup GitHub Pages (One-time):
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select **GitHub Actions**
3. Push to `master` — the workflow deploys automatically ✅
