# FinWise - Smart Finance Management for Every Indian Family

A modern, mobile-first personal finance web application for Indian users.

## Features
- 📊 Expense Tracker with AI auto-categorization
- 🧮 Tax Calculator (Old & New Regime FY 2025-26)
- 💰 Investment Tracker (SIPs, Mutual Funds, Stocks, Gold)
- 🛡️ Insurance Comparison (Health, Life, Term, Car, Bike)
- 🤖 AI Financial Advisor
- 🎯 Goal Planner & EMI Tracker
- 🌙 Dark/Light Mode
- 🇮🇳 Hindi, Gujarati & English support

## Tech Stack
- **Frontend**: Next.js 16, React, TypeScript
- **Styling**: Tailwind CSS + Custom CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **Font**: DM Sans + DM Serif Display

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Pages
- `/` — Landing page
- `/dashboard` — User financial dashboard  
- `/tax` — Tax calculator FY 2025-26
